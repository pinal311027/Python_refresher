## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Python** (version 3.8 or higher)  
  - Download from: https://www.python.org/downloads/

- **Jupyter Notebook** or **Jupyter Lab**  
  - Installation guide below

- **Git** (for cloning the repository)  
  - Download from: https://git-scm.com/

### Installation & Setup

#### Step 1: Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/python-refresher.git
cd python-refresher
```

Replace `YOUR-USERNAME` with your actual GitHub username.

#### Step 2: Create a Virtual Environment (Recommended)

**On macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**On Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

#### Step 3: Install Required Packages

```bash
pip install --upgrade pip
pip install jupyter notebook pandas numpy matplotlib
```

#### Step 4: Launch Jupyter Notebook

```bash
jupyter notebook
```

This will open Jupyter in your default browser. Navigate to the notebooks and start learning!

## 🐛 Troubleshooting

### Issue: Jupyter Notebook not starting

**Solution:**
```bash
pip install --upgrade jupyter
jupyter notebook --version
```

### Issue: Import errors when running notebooks

**Solution:**
```bash
pip install -r requirements.txt
```

### Issue: Kernel keeps crashing

**Solution:**
```bash
python -m ipykernel install --user
```

### Issue: Outdated Python version

**Solution:** Install Python 3.8+ from https://www.python.org/downloads/
